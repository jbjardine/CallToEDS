# Security Policy

## Reporting a Vulnerability
Please report security issues privately.

Contact:
- GitHub Security Advisories: https://github.com/jbjardine/CallToEDS/security/advisories
- Email: jbjardine@users.noreply.github.com

We aim to respond within 7 days and will coordinate a fix and disclosure timeline.

## Supported Versions
Security fixes are provided for the latest release only.

## Scope
This policy covers the Call2EDS codebase and official releases.
